Über mich...

Diese Webseite wurde von Tim Mattick, Matr.-Nr. 7322144, erstellt. Die Seite ist Teil einer Modulabschlussprüfung im Seminar „Advanced Web Basics”.

Auf dieser Seite sind einige Spieleklassiker der Arcade-Zeit bereitgestellt. Alle Spiele laufen in Javascript direkt im Browser. Zur Programmierung wurde die Bibliothek p5.js genutzt.

Ziel dieser Seite ist ist es, das Spielprinzip der Spieleklassiker zu zeigen. Daher wurde auf Game over-States, Spielmodi und weitere Annehmlichkeiten kein besonderer Wert gelegt, Ziel war die Umsetzung des Spielprinzips. Gleichzeitig ermöglicht eine solche Auseinandersetzung es, sich mit Logik und der praktischen Umsetzung einer Vielzahl von Problemen zu beschäftigen. Ich möchte daher jeden Interessierten ermutigen, sich selbst an die Programmierung einfacher Spiele zu wagen.
Der gesamte Quellcode der Seite ist auf GitHub verfügbar.
